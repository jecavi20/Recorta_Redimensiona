## 📌 Funciones de la app:

✅ Realizar recortes de pantalla desde el navegador

✅ Exporta el recorde en los formatos jpeg, png y webp

✅ Permite comprimir la imagen, para reducir el tamaño

✅ Puedes capturar una ventana, pestaña del navegador o pantalla completa

## 📌 Tecnologías
- HTML
- CSS
- JavaScript


## 📌 Instalación
1. Clonar o descargar el repositorio
2. Abrir el archivo index.html en el navegador
